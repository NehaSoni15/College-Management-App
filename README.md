# 📱 College Management App

## 🚀 Overview

A mobile application developed using Kotlin and SQLite to manage college operations such as student records, faculty data, and internal management.

---

## 🧰 Tech Stack

* Kotlin
* SQLite
* Android Studio

---

## ✨ Features

* Add / Update / Delete student records
* Manage faculty data
* Store data using SQLite database
* Simple and user-friendly interface
